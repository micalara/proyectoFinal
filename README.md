Este proyecto es una página web básica desarrollada como parte de un curso de Front-End. La página está estructurada con HTML semántico y utiliza las etiquetas <header>, <main>, y <footer> para organizar el contenido. El objetivo es aprender a crear la estructura básica de una página web y prepararla para futuras mejoras con CSS y JavaScript.
La pagina simula una tienda de papeleria la cual se puede elegir entre zona dark y zona rosa, basicamente es para separar los articulos entre los que son con esas tematicas, como para darle su toque a cada uno, ya sea todo minimalista y blanco y negro a la zona dark, y todo rosa con "brillitos" se podria decir, para la zona rosa.
Aunque tambien se puede elegir la opcion productos de la lista de navegacion para ver todos los productos sin esta separacion de zonas.
